# Konnected - Biosite

## 📖 Sobre o Projeto
Este é o repositório do Biosite da **Konnected**. O projeto consiste em uma Landing Page moderna e um conjunto de páginas de serviços e produtos criadas para ajudar comércios físicos e negócios locais a se destacarem no mercado digital. 

O site apresenta as soluções da empresa, que incluem:
- Plaquinhas e displays físicos inteligentes com tecnologia NFC.
- Biosites profissionais de carregamento ultrarrápido.
- Integração para agendamentos online e serviços de fotografia comercial.

A interface foi projetada com uma estética *Cyber/Neon*, utilizando temas escuros com tons de esmeralda brilhante para transmitir tecnologia, modernidade e resultados rápidos.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica e acessível.
- **Tailwind CSS (CDN):** Estilização ágil focada em design responsivo, animações e paleta de cores personalizada.
- **Google Fonts:** Tipografia moderna (Inter, Space Grotesk, Montserrat).

## 📁 Estrutura de Arquivos
- `index.html`: Página principal (Landing Page).
- `servicos-e-produtos.html`: Catálogo com os detalhes das soluções oferecidas.
- `pacotes-integrados.html`: Vitrine das opções de combos de serviços.
- `tabela-de-precos.html`: Planos e valores.
- `agendamento.html`: Interface com integração (embed) do Google Calendar para marcação de reuniões.

## 🛠️ Como rodar o projeto
Para visualizar o projeto localmente, basta fazer o download dos arquivos e abrir o `index.html` no seu navegador de preferência. Não é necessário instalar dependências ou usar servidores locais.

## 🌐 Deploy
Por ser composto integralmente de arquivos estáticos, o projeto pode ser facilmente hospedado em qualquer plataforma moderna de hospedagem gratuita, como **Vercel**, **Netlify** ou **GitHub Pages**.
